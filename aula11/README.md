# Aula 11 — Sistemas de Informações Geográficas (SIG) aplicado ao melhoramento genético

Data: 20/05/2026  
Carga horária: 4h

Professor convidado: Prof. Gustavo Eduardo Marcatti — UFSJ

## Conteúdo abordado

- Conceito de Sistemas de Informações Geográficas (SIG)
- Componentes básicos de um SIG: hardware, software e pessoas
- Representação do mundo real em dados espaciais
- Diferença entre dados vetoriais e dados raster
- Estruturas vetoriais: pontos, linhas e polígonos
- Estrutura raster: células, pixels e superfícies contínuas
- Principais formatos de arquivos vetoriais: shapefile, GeoPackage e GeoJSON
- Principais formatos de arquivos raster: TIFF, GeoTIFF, JPEG, JPEG2000 e ASCII
- Produção de dados espaciais por levantamento de campo, sensores proximais e sensoriamento remoto
- Fontes de dados ambientais: sensores ópticos, radar, LiDAR e dados meteorológicos
- Sistema de referência de coordenadas (SRC/CRS)
- Diferença entre coordenadas geográficas e coordenadas projetadas
- Aplicações de SIG no melhoramento genético
- Uso de SIG na ambientômica
- Obtenção de dados ambientais para modelagem G×A
- Geração de marcadores ambientômicos
- Predições ambientômicas em escala espacial
- Cubo de predições como produto da ambientômica

## Atividade

Discussão conceitual sobre o uso de SIG no melhoramento genético e na ambientômica.

- Identificar diferenças entre dados vetoriais e raster
- Reconhecer formatos básicos de arquivos geoespaciais
- Interpretar a importância do sistema de referência de coordenadas
- Relacionar dados ambientais espacializados com modelos de interação genótipos × ambientes
- Compreender o cubo de predições como produto operacional da ambientômica
